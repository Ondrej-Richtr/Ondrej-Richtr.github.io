2.5D Side-scroller Game

Year: 2023, Programmed in:  Zig

A 2.5D game experiment using Raylib.

Just a simple side-scrolling game test. I wanted to try out Zig, which is a new modern programing language that I think is pretty nice for games, in combination with Raylib.

During the short development of this project I realized that movement collisions are much easier if the player is able to move only on a grid. Also that drawing arbitrary 2.5D is not as easy as I thought, you can’t just layer sprites on top of eachother.

Download

You can download portable version for Windows x86_64 here (ZIP).

Note: Should work on Linux too when executed through Wine.

Controls

  • Arrows to move.
  • Space to jump.
  • Shift to attack the dummy.
  • Esc to exit the game.
